## ğŸ§  What is RFM Analysis?

RFM is a data-driven technique used to evaluate customer behavior based on three dimensions:

| Metric | Meaning | Interpretation |
|--------|----------|----------------|
| **Recency (R)** | Days since last purchase | Lower = more active |
| **Frequency (F)** | Number of transactions | Higher = more loyal |
| **Monetary (M)** | Total spending amount | Higher = more valuable |

Each customer gets an **RFM score**, which can then be used for segmentation, targeting, or clustering.
